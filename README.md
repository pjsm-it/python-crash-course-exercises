# Python Crash Course - Exercises & Projects

This repository contains exercises and projects inspired by the book *Python Crash Course* by Eric Matthes.  
All examples and exercises are rewritten in my own style and implemented from scratch. The book is used only as inspiration.

## Repository Structure

- `examples/` – Code examples written in my own style.  
- `exercises/` – Exercises implemented from scratch.  
- `projects/` – Larger projects inspired by the book, implemented in my own style.  

## Getting Started

1. Clone the repository:

git clone https://github.com/pjsm-it/python-crash-course.git

2. Navigate to the folder and run the Python files:

cd python-crash-course
python3 examples/example.py
